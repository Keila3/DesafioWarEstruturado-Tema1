# EstruturaDeDados-Desafios

Projeto em C que simula cadastro e conflitos entre territórios.

Funcionalidades
- Cadastrar territórios (nome, cor do exército, número de tropas).
- Exibir lista de territórios cadastrados.
- Validar entradas (quantidade, índices, tropas mínimas, cores).
- Simular ataque entre territórios usando rolagem aleatória de dados.
- Transferir tropas e atualizar propriedade em caso de conquista.
- Liberar memória alocada dinamicamente.

Tecnologias e ferramentas usadas
- Linguagem: C (padrão C11).
- Compilador: GCC.
- Bibliotecas padrão: stdio.h, stdlib.h, string.h, time.h.
- Ambiente de desenvolvimento: Visual Studio Code (tasks.json / launch.json para build/depuração).
- Execução/entrada interativa via Terminal integrado (ou terminal do sistema).
